# CLAUDE.md

Personal academic website for Vivek Anand, built with Zola + Typst. Deployed on Netlify at vivek-anand.xyz.

## Architecture

- **Zola** — static site generator (config in `config.toml`, deploys via `netlify.toml`)
- **Typst** — blog posts are authored as `.typ` files, converted to HTML via Pandoc
- **SCSS** — styling in `sass/`, uses CSS custom properties for dark/light theming
- **BibTeX** — publications sourced from `publications.bib`, converted to Zola content pages

## Build & Dev Commands

```powershell
# Local dev server with hot reload
zola serve

# Convert all .typ files to HTML (must run before zola build if Typst content changed)
.\build.ps1

# Production build (also what Netlify runs)
zola build

# Create a new blog post
.\scripts\new-post.ps1 -Title "Post Title" -Tags "tag1, tag2"

# Regenerate publication pages from BibTeX
powershell -ExecutionPolicy Bypass -File .\scripts\bib-to-zola.ps1 -BibFile publications.bib
```

## Content Workflow

### Blog posts
Each post lives in `content/blog/<slug>/` with:
- `index.md` — Zola frontmatter (title, date, tags, series nav)
- `content.typ` — Typst source
- `content.html` — generated by `build.ps1` (Pandoc: typst -> HTML with --mathjax --citeproc)
- `refs.bib` — optional bibliography for citations

The `{% typst() %}` shortcode in `index.md` loads the generated HTML.

### Publications
1. Edit `publications.bib` (supports custom fields: `tags`, `paper_url`, `poster_url`, `slides_url`, `code_url`, `video_url`, `website_url`, `blog_url`, `award`)
2. Run `bib-to-zola.ps1` to regenerate `content/publications/*.md`
3. Local PDFs go in `static/files/`

### Homepage
Edit `content/_index.md` for bio paragraphs (`[[extra.bio_paragraphs]]`), education (`[[extra.education]]`), and social links.

## Key Directories

```
content/          # Markdown + Typst source content
templates/        # Zola Jinja2 templates
  partials/       # Reusable template fragments (pub-buttons.html)
  shortcodes/     # typst.html shortcode
sass/             # SCSS (variables, base, layout, components)
static/files/     # PDFs, CSL styles
scripts/          # PowerShell utilities
```

## Conventions

- Templates use BEM-style CSS classes (e.g., `.publications__item`, `.hero__greeting`)
- Author name "Anand, Vivek" is auto-bolded in publication listings
- Tags are shared across blog posts and publications
- Dark theme is the default; light theme via `[data-theme="light"]`
- Series navigation uses `series_prev`/`series_next` in post frontmatter `[extra]`
- The site uses Inter font family and MathJax 3 for math rendering
